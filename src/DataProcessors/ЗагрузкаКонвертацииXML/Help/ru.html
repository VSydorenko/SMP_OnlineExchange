Обработка "Загрузка конвертации XML"

Предназначена для загрузки конвертации настроенной в формате XML 
Имя файла правил: указывается файл с расширением xml, содержащий правила конвертации
Варианты загрузки:
  • Создать новую конвертацию. При загрузке будет создана новая конвертация, ее параметры будут заполнены в соответствии с данными из файла xml. В диалоге потребуется указать Наименование конвертации, Конфигурацию источник и Конфигурацию получатель.
  • Заменить правила конвертации правилами из файла. В диалоге потребуется указать Конвертацию. Правила конвертации будут замещены.
  • Дополнить правила конвертации правилами из файла. В диалоге потребуется указать Конвертацию. Будут добавлены недостающие правила.
